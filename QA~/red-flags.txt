# SDK/vendor red-flag substrings — these MUST NOT appear in a greenlight run, and any
# hit is BINDING (it halts the run). Scope: SDK and vendor init/delivery failures only.
# A game's own exceptions and content errors are NOT here: they are informational, they
# live in game-exceptions.txt, and they never fail an SDK-scoped run (a game with its own
# NullReferenceExceptions can still have a completely healthy SDK integration).
# One substring per line. Lines starting with # are ignored.
# Grep list the agent applies to a RAW capture (iOS devicectl /tmp/unity_ios_rolling.log
# or Android logcat). No capture script auto-loads it (pymobiledevice3/ios_logs.py retired
# cycle 0014). Usage:  grep -Ff <(grep -v '^#' protocols/red-flags.txt | grep -v '^$') <capture>
# (strip comments/blanks first: grep -Ff treats every line, including these comments,
#  as a pattern; raw -Ff false-fired on 2 innocuous lines in the 2026-06-12 hungrysnake run)
